# Изображения инструкции Canon R6

Эта папка содержит изображения для пошаговой инструкции настройки фотоаппарата Canon R6.

## Структура файлов

### Настройка именования файлов:
- `naming-step-1.jpg` - Переход в меню настроек имени файла
- `naming-step-2.jpg` - Выбор пользовательской настройки
- `naming-step-3.jpg` - Ввод префикса (cameraId)
- `naming-step-4.jpg` - Выход из режима настройки
- `naming-step-5.jpg` - Выбор зарегистрированного имени файла

### Настройка FTP подключения:
- `ftp-step-1.jpg` - Включение Wi-Fi
- `ftp-step-2.jpg` - Выбор беспроводных функций
- `ftp-step-3.jpg` - Выбор передачи на FTP-сервер
- `ftp-step-4.jpg` - Добавление устройства для подключения
- `ftp-step-5.jpg` - Выбор метода настройки
- `ftp-step-6.jpg` - Подключение к точке доступа
- `ftp-step-7.jpg` - Настройка IP-адреса
- `ftp-step-8.jpg` - Настройка IPv6
- `ftp-step-9.jpg` - Выбор режима FTP
- `ftp-step-10.jpg` - Настройка адреса сервера FTP
- `ftp-step-11.jpg` - Ввод IP-адреса сервера FTP
- `ftp-step-12.jpg` - Настройка номера порта
- `ftp-step-13.jpg` - Ввод имени пользователя FTP
- `ftp-step-14.jpg` - Ввод пароля FTP
- `ftp-step-15.jpg` - Настройка папки назначения
- `ftp-step-16.jpg` - Настройки передачи по FTP

### Дополнительно:
- `placeholder.jpg` - Заглушка для отсутствующих изображений

## Источники изображений

Изображения можно скачать с официального сайта Canon:
- https://cam.start.canon/ru/C004/manual/html/UG-07_Set-up_0060.html (настройка именования)
- https://cam.start.canon/ru/C004/manual/html/UG-06_Network_0070.html (настройка FTP)
